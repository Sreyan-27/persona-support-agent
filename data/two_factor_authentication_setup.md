# Two-Factor Authentication (2FA) Setup

## Overview
Two-factor authentication adds a second verification step to your TechCorp Cloud login, using a time-based one-time code (TOTP) from an authenticator app.

## Enabling 2FA
1. Go to Account Settings → Security.
2. Click "Enable Two-Factor Authentication."
3. Scan the displayed QR code using an authenticator app (Google Authenticator, Authy, or 1Password).
4. Enter the 6-digit code generated by the app to confirm setup.
5. You will be shown 10 single-use backup codes. Download or print these and store them somewhere safe — they are the only way back into your account if you lose your authenticator device.

## Disabling 2FA
1. Go to Account Settings → Security → Two-Factor Authentication.
2. Click "Disable."
3. Confirm with your current password and a valid 6-digit code.

## Lost Authenticator Device
- Use one of your 10 backup codes at the login screen by selecting "Use a backup code."
- Each backup code can only be used once. After use, it is automatically removed from your list.
- If all backup codes are exhausted and the device is lost, self-service recovery is not possible. This requires manual identity verification by a human support agent and typically takes 1–2 business days.

## Enterprise Accounts
Enterprise-tier admins can enforce mandatory 2FA for all workspace members under Account Settings → Security → Organization Policies. When enforced, users without 2FA configured will be prompted to set it up on next login.
